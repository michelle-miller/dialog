# flow
{: #flow}

The `<flow>` node is a container for the main content of a dialog flow.
{: shortdesc}

## Example

```xml
<flow>
  <folder label="Main"></folder>
  <folder label="Library"></folder>
  <folder label="Global"></folder>
  <folder label="Concepts"></folder>
</flow>
```
{: codeblock}
