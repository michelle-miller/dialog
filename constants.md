# constants
{: #constants}

The `<constants>` node stores variables for information that might change over time, such as product names or prices.
{: shortdesc}

## Example

```xml
<constants>
  <var_folder>
    <var name="VA" type="TAG">Virtual Agent</var>
  </var_folder>
</constants>
```
{: codeblock}
