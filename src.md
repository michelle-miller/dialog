# src
{: #src}

The `<src>` node is mapped to a `<tgt>` node in a translation. The text in `<src>` is translated to the text in `<tgt>`.
{: shortdesc}

## Example

```xml
<translations>
  <translation>
    <src>:)</src>
    <tgt>smile</tgt>
  </translation>
</translations>
```
{: codeblock}
