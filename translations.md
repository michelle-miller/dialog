# translations
{: #translations}

The `<translations>` node contains sets of translations.
{: shortdesc}

## Example

```xml
<translations>
  <translationSet>
    ^^ ~ smile
    ^_^ ~ smile
    - ~ subtract
    ; - ) ~ wink
    ; ) ~ wink
    ;_; ~ sad
    ;-) ~ wink
    ;-)) ~ wink
    ;-))) ~ wink
    ;-p ~ stick out tongue
    ;) ~ wink
    ;)) ~ wink
    ;))) ~ wink
    ;o) ~ wink
    ;op ~ stick out tongue
    ;opp ~ stick out tongue
    ;oppp ~ stick out tongue
    ;p ~ stick out tongue
    : ( ~ sad
    : ) ~ smile
    : ] ~ smile
    : d ~ smile
    : o ~ surprise
    : p ~ stick out tongue
    :-( ~ sad
    :-(( ~ sad
    :-((( ~ sad
    :-) ~ smile
    :-)) ~ smile
    :-))) ~ smile
    :-d ~ smile
    :-dd ~ smile
    :-ddd ~ smile
    :-o ~ surprise
  </translationSet>
</translations>
```
{: codeblock}
