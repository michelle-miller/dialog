# specialSettings
{: #specialSettings}

The `<specialSettings>` node contains the child nodes for several specialized settings.
{: shortdesc}

## Example

```xml
<specialSettings>
  <specialSetting label="AutoLearn Statement" selectionType="RANDOM">
    <variations>
      <item>Did you mean?</item>
    </variations>
  </specialSetting>
</specialSettings>
```
{: codeblock}
