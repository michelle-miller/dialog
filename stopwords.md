# stopwords
{: #stopwords}

The `<stopwords>` node contains `<stopwordSet>` nodes.
{: shortdesc}

## Example

```xml
<stopwords>
  <stopwordSet>
    after
    afterward
    against
  </stopwordSet>
  <stopwordSet>
    also
    although
    always
  </stopwordSet>
</stopwords>
```
{: codeblock}
